# МедКлиника — административная панель медицинской клиники

Полноценный учебный проект для ПМ.08: Flask + MySQL, авторизация, роли пользователей, dashboard, CRUD-разделы, поиск, журнал действий и адаптивный интерфейс.

Предметная область проекта — медицинская клиника.

---

## Что реализовано

- вход и выход пользователей;
- роли `admin`, `manager`, `viewer`;
- dashboard со статистикой и последними действиями;
- CRUD-разделы:
  - врачи;
  - пациенты;
  - услуги;
  - записи на приём;
- поиск врачей, пациентов, услуг и записей;
- управление пользователями;
- блокировка пользователей;
- журнал действий;
- подтверждение удаления;
- flash-уведомления;
- адаптивный интерфейс;
- тестовые данные для демонстрации.

## Роли пользователей

### admin
Полный доступ к системе:
- просмотр данных;
- добавление данных;
- редактирование данных;
- удаление данных;
- управление пользователями;
- изменение статусов пользователей;
- просмотр журнала действий.

### manager
Доступ к рабочим разделам:
- просмотр данных;
- добавление данных;
- редактирование данных;
- работа с записями;
- без удаления данных;
- без доступа к управлению пользователями.

### viewer
Режим просмотра:
- просмотр информации;
- просмотр dashboard;
- просмотр журналов;
- без возможности изменения данных.

## Запуск проекта

1. Установить Python и XAMPP.

2. Запустить Apache и MySQL в XAMPP.

3. Создать базу данных `clinic_db` через phpMyAdmin.

4. Импортировать файл `database/clinic.sql`

5. Установить библиотеки:

```bash
pip install flask
pip install flask-mysqldb
```

6. Запустить проект:

```bash
python app.py
```

7. Открыть в браузере:

`http://127.0.0.1:5000`
